# 🛍️ 商品外卖平台链接功能完整实现

## ✅ 已完成功能

### 1. 后台管理 - 添加商品时可配置外卖链接

**路径**: `/admin/products`

#### 使用流程

**步骤 1: 创建新商品**
1. 访问 `/admin/products`
2. 点击"+ 添加商品"
3. 填写商品信息（名称、价格、分类等）
4. 点击"创建"按钮

**步骤 2: 添加外卖平台链接**
- ✅ **创建成功后，表单不会关闭**
- ✅ **页面底部自动显示"外卖平台链接"管理区域**
- ✅ 可以立即为该商品添加多个外卖平台链接

**步骤 3: 编辑已有商品**
- 点击商品列表中的"编辑"按钮
- 修改商品信息后保存
- 或直接滚动到页面底部管理外卖链接

## 🗄️ 数据库表结构

### product_waimai_links 表

```sql
CREATE TABLE product_waimai_links (
  id UUID PRIMARY KEY,
  product_id UUID NOT NULL,           -- 关联的商品ID
  platform_key TEXT NOT NULL,         -- 平台标识：meituan, eleme, etc.
  platform_name TEXT NOT NULL,        -- 平台名称（可自定义）
  platform_icon TEXT,                 -- 图标 emoji
  shop_url TEXT NOT NULL,             -- 店铺首页链接
  product_url TEXT,                   -- 商品详情页链接（可选）
  is_enabled BOOLEAN DEFAULT true,    -- 是否启用
  sort_order INTEGER DEFAULT 0,       -- 排序顺序
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ
)
```

## ⚙️ 高级配置

### 自定义平台名称

**示例**:
- 标准名称：美团外卖
- 自定义：黄老牛肉干美团旗舰店

**操作**: 在添加平台时，修改"平台名称"字段

---

### 自定义图标

**示例**:
- 默认：🟡
- 自定义：🍖 或 🏪

**操作**: 在添加平台时，修改"图标 Emoji"字段

---

### 设置商品专属链接

如果某个商品有独立的商品详情页链接（不是店铺首页）：

**店铺链接**: `https://meituan.com/shop/123`
**商品链接**: `https://meituan.com/shop/123/product/456`

**操作**: 填写"商品详情页链接"字段，优先使用该链接

---

### 排序控制

通过 `sort_order` 字段控制显示顺序：
- 数字越小越靠前
- 默认为 0
- 可以在数据库中直接修改

**SQL 示例**:
```sql
UPDATE product_waimai_links 
SET sort_order = 1 
WHERE product_id = '商品ID' AND platform_key = 'wechat';
```

## 🔍 常见问题

### Q: 创建商品后看不到外卖链接管理？

**A**: 
1. 确认商品创建成功（没有报错）
2. 刷新页面
3. 检查控制台是否有错误
4. 确保 `product_waimai_links` 表已创建

---

### Q: 前台不显示购买渠道按钮？

**A**: 检查以下几点：
1. 是否已为该商品添加了外卖链接
2. 链接的 `is_enabled` 是否为 true
3. `shop_url` 是否已填写
4. 刷新页面缓存

---

### Q: 可以为同一个平台添加多个链接吗？

**A**: 可以！比如你可以添加两个美团链接（不同店铺）。系统不会限制。

---

### Q: 如何删除已添加的外卖链接？

**A**: 
1. 在后台编辑商品
2. 滚动到"外卖平台链接"区域
3. 找到要删除的链接
4. 点击右侧的"删除"按钮

---

### Q: 如何临时禁用某个平台？

**A**: 
1. 在后台编辑商品
2. 找到对应的外卖链接
3. 关闭"启用此平台"开关
4. 前台将不再显示该按钮
